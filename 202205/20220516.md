# MarkDown
## 1. MarkDown이란  
    텍스트 기반의 경량 마크업 언어로 문법이 쉽고 간단함.

## 2. 문법  
### 2.1. 제목  
    #문자를 1~6개 붙여서 제목 표현 가능  
    # 제목1  
    ## 제목2
    ### 제목3
    #### 제목4
    ##### 제목5
    ###### 제목6
> # 제목1
> ## 제목2
> ### 제목3
> #### 제목4
> ##### 제목5
> ###### 제목6

### 2.2. 인용문  
    > 블럭인용문자를 사용하며 블럭 안 다른 마크다운 요소도 사용 가능
    > 첫번째
    >> 2. 두번째
    >>> + 세번째
> 첫번째
>> 2. 두번째
>>> + 세번째

### 2.3. 목록  
#### 2.3.1. 순서있는 목록  
    순서있는 목록은 숫자와 마침표를 사용(숫자 상관없이 자동으로 정렬됨)
    1. 첫번째
       1. 1번
       4. 2번
    2. 두번째
    1. 세번째
1. 첫번째
   1. 1번
   4. 2번
2. 두번째
1. 세번째

#### 2.3.2. 순서없는 목록  
    순서없는 목록은 기호 *, -, + 를 사용(혼합해서 사용도 가능함)
    * 첫번째
       * 1-1
       * 1-2
       * 1-3
    + 두번째
       - 2-1
       - 2-2
    - 세번째
       * 3-1
          * 3-1-1
* 첫번째
   * 1-1
   * 1-2
   * 1-3
+ 두번째
   - 2-1
   - 2-2
- 세번째
   * 3-1
     * 3-1-1
