# JavaScript
* 즉시 실행 함수
  - 함수 **정의와 동시에 즉시 실행**되는 함수
  - 단 한 번만 호출되면 다시 호출 불가
  - 기명함수도 가능하지만 익명함수 사용이 일반적
  - 반드시 그룹 연산자 ('()')로 감쌀 것 → 함수 리터럴로 평가, 함수 객체가 됨
  - 함수 리터럴을 평가해서 함수 객체 생성 가능하면 그룹 연산자 이외 연산자도 사용 가능
    > 방법 1. (function () {});  
    > 방법 2. (function () {})();  
    > 방법 3. !function () {} ();  
    > 방법 4. +function () {} ();
  - 값 반환과 인수 전달도 가능
  - 즉시 실행 함수 내에 코드를 모아 두면 변수나 함수 이름 충돌 방지 가능
